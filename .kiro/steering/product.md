# Product Overview

This is a personal NixOS and Home Manager configuration repository that provides modular, cross-platform system configurations.

## Purpose
- Manage consistent development environments across multiple hosts and platforms
- Support both NixOS systems and Home Manager configurations
- Enable reproducible setups for desktop, server, and work environments

## Key Features
- **Multi-platform support**: Linux (x86_64) and macOS (aarch64)
- **Environment-specific configurations**: desktop, server, work setups
- **Hyprland desktop environment** with custom theming
- **WSL support** for Windows development
- **Secure Boot** integration via Lanzaboote
- **Development toolchain** with consistent shell and editor configurations

## Target Users
Personal use configuration for maintaining consistent development environments across different machines and operating systems.
